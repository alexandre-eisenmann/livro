\begin{ex}
 Um experimento aleatório tem espaço amostral equiprovável E, finito e não vazio. Se um evento A desse espaço amostral é tal que $P(A)=\frac{n-5}{8}$ , então:
   \begin{enumerate}[(a)]
   \item $ 5 \leq  n \leq  13 $
   \item $ 5 < n < 13 $
   \item $ n = 13 $
   \item $ n = 5 $
   \item $ n = 0 $
   \end{enumerate}
     \begin{sol}
      resposta: a \\
      $0\leq\frac{n-5}{8}\leq1 \Longrightarrow 5\leq n \leq13$
     \end{sol}
\end{ex}
