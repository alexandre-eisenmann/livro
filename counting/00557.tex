\begin{ex}
Um dado é lançado três vezes consecutivas. Qual é a probabilidade de que a soma dos números de pontos nas faces voltadas para cima:
   \begin{enumerate}[(a)]
   \item ultrapase 18?
   \item seja menor que 19?
   \end{enumerate}
     \begin{sol}
      \phantom{A}
      \begin{enumerate} [(a)]
          \item $p=0$
          \item $p=1$
      \end{enumerate}
     \end{sol}
\end{ex}
